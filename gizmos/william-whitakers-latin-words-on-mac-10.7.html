The official OS X version of William Whitaker’s Words doesn’t work with OS X 10.7 (Lion). A google search indicates that William Whitaker, the creator the software, passed away in 2010.

Thankfully, the program is open source. So I’ve recompiled it as a universal binary:

  • Command Line Only Version of Words

Source for the official GUI is not available. But Erik Mendoza has updated his GUI wrapper,

  • Interpres.app,

to include my new binary.

If you want a GUI version, try this:

  • Unofficial Hacked Copy of Interpres.app for 10.7

(Interpres.app is GUI wrapper around Words, written by Erik Mendoza. The official version, which does not work with Lion, is available here.)
