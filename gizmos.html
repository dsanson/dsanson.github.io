I maintain several small coding projects, mostly related to John MacFarlane's pandoc, which are all hosted on github. These include:

  • Pandoc-Droplets-and-Services: a set of drag-and-drop applets and system services for using pandoc on OS X, for those afraid of the command line.
  • vim-pandoc: a plugin for writing pandoc's flavor of markdown in [Vim] text editor.
  • Pandoc.tmbundle: a plugin for writing pandoc's flavor of markdown in [TextMate].
  • bibdesk-pandoc-export-templates: templates for use with BibDesk, enabling drag-and-drop support for pandoc-style citations.
