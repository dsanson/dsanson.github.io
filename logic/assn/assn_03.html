Instructions

This is an assignment to be completed on paper, and turned in in class. All problems are worth 10 points each, and will be added to the points you have earned on the quizzes and the Logic 2010 homework assignments.

Symbolization

  1. ‘unless’ is another way of saying _______?
  2. How do we symbolize ‘Every A is B’?
  3. How do we symbolize ‘Some A is B’?
  4. How do we symbolize ‘Only As are Bs’?
  5. How do we symbolize ‘No A is B’?
  6. Explain why ‘\({\exists}x(Ax{\mathbin{\rightarrow}}Bx)\)’ can be true when ‘Some As are Bs’ is false, and so is not the right way to symbolize ‘All As are Bs’.
  7. Explain why ‘\({\forall}x(Ax{\mathbin{\wedge}}Bx)\)’ can be false when ‘All As are Bs’ is true, and so is not the right way to symbolize ‘All As are Bs’.

Arguments

For each of the following arguments, complete the following tasks:

  1. Think through the argument without the help of our tools, and decide whether or not you think it is valid. Write down your answer (it is okay if this turns out to be wrong).
  2. Symbolize the argument (10 pts).
  3. If the argument is invalid, construct a model that shows this (10 pts).
  4. If the argument is valid, construct a derivation that shows this (10 pts).
  1. All dogs go to heaven. Fido is a dog. So Fido goes to heaven.
  2. Only dogs go to heaven. Fido is a dog. So Fido goes to heaven.
  3. All dogs go to heaven. So everything that goes to heaven is a dog.
  4. Some dogs are evil. Nothing evil goes to heaven. So some dogs don’t go to heaven.
