Instructions

This is an assignment to be completed on paper, and turned in in class. All problems are worth 10 points each, and will be added to the points you have earned on the quizzes and the Logic 2010 homework assignments.

Multiple Choice

  1. If an argument has false premises and a true conclusion, we know that:
    1. it is valid
    2. it is not valid
    3. it is sound
    4. it is not sound
    5. none of the above
  2. If an argument has true premises and a true conclusion, we know that:
    1. it is valid
    2. it is not valid
    3. it is sound
    4. it is not sound
    5. none of the above
  3. If an argument has true premises and a false conclusion, we know that:
    1. it is valid
    2. it is not valid
    3. it is sound
    4. it is not sound
    5. none of the above

Validity and Soundness

Complete the following definitions:

  1. An argument is valid just in case ______?

  2. An argument is sound just in case ______?

Symbolize and Derive

Symbolize each of the following arguments (10 pts), and then show that they are valid by constructing derivations (10 pts):

  1. We won’t defeat the terrorists if we lose our moral authority. Moreover, if we torture detainees, we lose our moral authority. So, we won’t defeat the terrorists if we torture detainees.

  2. Humans don’t share a common language. If there were a natural connection between words and the ideas they stand for, all humans would share a common language. Moreover, the connection between words and the ideas is arbitrary if it is not natural. So the connection between words and the ideas for which they stand is arbitrary.
